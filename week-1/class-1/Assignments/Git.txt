Tasks:

Git and Github

1. Create a new directory, initialize a Git repository, and create an initial commit with a README file.
2. Familiarize yourself with essential Git commands like git add, git commit, git status, and git log.
3. Fork a public repository on GitHub and clone it to your local machine.
4. Make changes in your local repository and push them to your forked remote repository.
5. Create a pull request to propose changes to the original repository.
